# Certificate Gallery

This repository contains a collection of certificates intended for display purposes. The certificate gallery showcases templates and examples of various certificates that can be used for diverse events.

## Project Structure

```
DBKeeper/
└── certificates/
    ├── templates/
    │   ├── certificate_template1.pdf
    │   └── certificate_template2.pdf
    ├── images/
    │   ├── certificate_image1.png
    │   └── certificate_image2.png
    └── README.md
```

- **templates/**: Contains the PDF templates of the certificates.
- **images/**: Contains images used for displaying certificates in the gallery.

## GitHub Pages Setup

To set up GitHub Pages for this repository, follow these steps:

1. Go to the repository settings.
2. Scroll down to the **GitHub Pages** section.
3. Under **Source**, select the branch you want to use, usually `main`.
4. Click **Save**.
5. Your GitHub Pages will be published at `https://<username>.github.io/<repository-name>/`.

Replace `<username>` with your GitHub username and `<repository-name>` with the name of this repository.
